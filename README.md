# Profissional Destaque do Mês - Studio X

Dashboard de acompanhamento de desempenho dos profissionais do Studio X, com rankings por categoria (Cabelo, Unhas, Estética, Maquiagem), painel de premiação e integração com Google Stars.

## Tech Stack

- React 18 + TypeScript
- Vite
- shadcn/ui + Tailwind CSS
- Supabase (PostgreSQL)
- Recharts

## Desenvolvimento

```bash
npm install
npm run dev      # porta 8080
npm run build    # build de produção
npm run lint     # linting
```
